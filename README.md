# Anime Moments Web App

A **sleek and interactive web application** that brings your favorite anime moments to life.  
The app fetches real-time data from the **Jikan API** (for anime information) and a **Quotes API** (for character quotes), providing users with a dynamic and engaging anime experience. Built with a focus on **clean design, smooth interactions, and modern frontend development practices**.
